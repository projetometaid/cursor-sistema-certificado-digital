Visualizar Fila de Eventos
O sistema irá possibilitar a exibição do log de eventos enviados, bem como possibilitar o filtro por status, conforme imagem a seguir.

Gerenciamento AC.

Atributos de Entrada:

1. Autoridade Certificadora.
2. Autoridade de Registro.

6. Tipo de Evento.
7. Status do Evento.

1. Protocolo.
2. Data do Evento.
3. No Tentativas.

Overview
Para consumir o será necessário incluí-lo em sua aplicação através do link de acesso. O restante do processo será descrito nas próximas seções.

O URL de acesso ao Serviço será definido na instalação do mesmo ambiente da AC.

Atenção! O parâmetro “Authorization” é definido pelo método Autenticação.

Atenção!

- Para “Adicionar Solicitações Presenciais” você deve chamar o método Presencial.
- Para “Adicionar Solicitações de Renovação Própria” você deve chamar o método Renovação própria.
- Para “Adicionar Solicitações de Renovação de outra AC você deve chamar o método Renovação de outra AC.
- Para “Adicionar Solicitações de Videoconferência” você deve chamar o método Videoconferência.
